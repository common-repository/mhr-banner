== Description ==

Publish your banner/advertisement in unique way, on page footer.Support all images format including bmp,jpeg,
jpg,png,gif,etc.


= Please note = 
* After activate this plugin please goto *Settings* menu and select enable MHR-Banner to enable .
* For Banner URL you must use full address with 'http://'. 
* Make sure your banner width setting did not exceed your width screen.


== Installation ==

1. Upload the entire folder of MHR-Banner to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to **Settings** and then **MHR-Banner** to configure and enable.
